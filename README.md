# mybatisPlusCodeGenerator
嫌弃自己每次代码生成的时候都要改这个改那个，麻烦得很，于是写了个gui界面，只需要自己改一下数据库地址和表名，然后自动扫描数据库内的所有表，选择对应的表命就能自动生成代码了


主要办法就是通过填入地址，端口号，数据库名，用户名，密码这些数据来扫描 输入数据库 内的所有表，并弹出窗口展示，选择对应表名之后，点击生成代码，就可以生成对应的代码。
